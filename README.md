## Installation
#### via curl
`sh -c "$(curl -fsSL https://raw.githubusercontent.com/lucasscariot/config/master/setup.sh)"`

#### via wget
`sh -c "$(wget https://raw.githubusercontent.com/lucasscariot/config/master/setup.sh -O -)"`
